# ReportAssist

ReportAssist is a web platform for citizens to report non-emergency crimes to the police via an AI-driven interview process.

## Project Overview

The system is designed to manage reports through a lifecycle: Citizen Submission -> Triage Review -> Officer Assignment.

## Technical Stack

*   **Frontend**: Vanilla HTML5, CSS3, and modern Asynchronous JavaScript (ES6+).
*   **Backend**: Node.js with Express.js.
*   **Database**: Relational (PostgreSQL/MySQL).
*   **Authentication**: Role-Based Access Control (RBAC).

## Getting Started

1.  Install dependencies: `npm install`
2.  Set up your `.env` file with database credentials.
3.  Run database migrations.
4.  Start the server: `npm start`
